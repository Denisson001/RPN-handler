## Usage example

- Собрать проект (директория **src**)

```
g++ main.cpp data.cpp solver.cpp -std=c++11 -o main
```

- Собрать и запустить тесты (директория **tests**)

```
g++ tests.cpp ../src/solver.cpp -std=c++11 -o tests
./tests
```
